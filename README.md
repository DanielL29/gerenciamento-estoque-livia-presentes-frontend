# Gerenciamento Estoque Livia Presentes - Frontend

Frontend React para o sistema de estoque da Livia Presentes.

## Variaveis de ambiente

Copie `.env.example` para `.env` em desenvolvimento:

```env
VITE_API_URL=http://localhost:4000
```

Na Vercel, configure `VITE_API_URL` com a URL publica do backend no Render.

## Rodando local

```bash
npm install
npm start
```

O app sobe em `http://localhost:3000`.

## Build

```bash
npm run build
```

## Primeiro acesso

Com o banco vazio, use a tela de cadastro. O backend transforma o primeiro usuario cadastrado em administrador automaticamente.
